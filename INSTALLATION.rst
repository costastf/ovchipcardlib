============
Installation
============

At the command line::

    $ pip install ovchipcardlib

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv ovchipcardlib
    $ pip install ovchipcardlib

Or, if you are using pipenv::

    $ pipenv install ovchipcardlib

Or, if you are using pipx::

    $ pipx install ovchipcardlib
